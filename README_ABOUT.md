# 🚀 Last Minute Life Saver

> An AI-powered productivity companion that helps users plan, prioritize, and complete tasks before deadlines are missed.

## 📌 Problem Statement

Students, professionals, and entrepreneurs often miss assignments, meetings, bill payments, and important commitments because traditional reminder apps only provide passive notifications that are easy to ignore.

**Last Minute Life Saver** solves this problem by acting as an intelligent productivity companion that proactively assists users in taking action before deadlines.

---

## ✨ Features

* 🧠 AI-powered task prioritization
* 📅 Smart deadline and schedule management
* 🔔 Intelligent reminders and notifications
* 💬 Personalized productivity guidance
* 📊 Progress and task tracking
* ⚡ Proactive recommendations instead of passive reminders
* 📱 User-friendly interface built with Google Gemini AI Studio

---

## 💡 How It Works

1. User adds tasks and deadlines.
2. AI analyzes urgency and importance.
3. Tasks are prioritized automatically.
4. The system generates actionable plans and reminders.
5. Users receive guidance to complete tasks before deadlines.
